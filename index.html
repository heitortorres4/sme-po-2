Como usar o sistema

Passo 1 — Escolas:Cadastre as escolas informando modalidade(s) (pode marcar mais de uma), turno e turmas com seus turnos individuais.

Passo 2 — Professores:Cadastre cada professor com sua disciplina, e defina para cada escola/turma quantas aulas ele dá.

Passo 3 — Afastamentos:Registre afastamentos com motivo, data de início e previsão de retorno. O sistema identifica automaticamente as aulas que ficam livres.

Passo 4 — Painel e Aulas Livres:Atualizam em tempo real.

Logo:Clique no espaço de logo na barra lateral para carregar a imagem. Ela fica salva no navegador.

Atenção:Exporte o backup regularmente. Limpar o cache do navegador apaga os dados.

Como colocar o sistema online

Para que outras pessoas acessem sem precisar baixar o arquivo, você pode hospedar o HTML em uma das opções abaixo:

GitHub Pages — Gratuito e recomendado

Crie uma conta gratuita em github.com, crie um repositório, faça upload do arquivo HTML com o nome index.htmle ative o GitHub Pages nas configurações. O link será algo como seunome.github.io/sistema-aulas.

Netlify Drop — Mais fácil ainda

Acesse app.netlify.com/drop, arraste o arquivo HTML direto para a tela e em segundos você tem um link público. Sem precisar criar conta.

Google Drive — Para uso interno

Faça upload no Google Drive e compartilhe o link com a equipe. Cada pessoa precisará baixar o arquivo e abrir no navegador — não é um link direto na web, mas funciona para uso interno.

Importante sobre os dados

Cada pessoa que abrir o sistema no próprio computador terá seus próprios dados locais. Para que todos vejam os mesmos dados, a melhor solução futura é migrar para um banco de dados online. Me avise quando quiser evoluir para isso.
